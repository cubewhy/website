# 自定义API

## 支持的(类)API后端

1. [LunarCN API](https://github.com/CubeWhyMC/lunarclient.top) (完整支持)
2. [Moonsworth API](https://api.lunarclientprod.com) (部分支持,详细见[此处](#与moonsworth-api的兼容性))

## 与Moonsworth API的兼容性

1. 不支持插件市场
2. 不支持错误报告上传

## 修改API

> 输入完成之后记得按下enter进行保存

1. 打开Celestial
2. 打开设置页面
3. 找到API
4. 填写API即可,重启生效
